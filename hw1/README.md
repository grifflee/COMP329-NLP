# HW1 

## How to Run

```bash
python3 hw1.py
```

## Required Files

The following data files must be in the same directory as hw1.py:
- `rt-polarity.pos` - Positive movie review sentences
- `rt-polarity.neg` - Negative movie review sentences

## Output

The script outputs the accuracy of the binary sentiment classifier when tested on the positive and negative review files
