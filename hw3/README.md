# HW3

## How to Run

```bash
python3 hw3.py
```

## Required Files

The following data files must be in the same directory as hw3.py:
- `rt-polarity.pos` - Positive movie review sentences
- `rt-polarity.neg` - Negative movie review sentences

## Dependencies

```bash
pip install numpy scikit-learn
```

## Output

The script trains and evaluates three Logistic Regression sentiment classifiers using different vectorization methods
