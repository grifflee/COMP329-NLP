# Homework 2 - Naive Bayes Sentiment Classifier

## How to Run

```
python homework2.py
```

## Requirements

- Python 3
- No external libraries required (only uses `math` and `random` from the standard library)

## Data Files

The following data files must be in the same directory as `homework2.py`:

- `rt-polarity.pos` — positive movie reviews
- `rt-polarity.neg` — negative movie reviews

## Expected Output

The script will print:

1. Development set accuracy (training data only)
2. Test set accuracy (training + development data)
3. Example reviews with classifier confidence scores
